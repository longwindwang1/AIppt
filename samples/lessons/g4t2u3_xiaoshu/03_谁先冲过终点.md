# 谁先冲过终点（北师大版四年级下册 第三单元 第 3 课时）

## 教学目标

- 在跑步比赛情境中应用小数加减法解决实际问题
- 会进行小数加减混合运算
- 理解"用时少 = 跑得快"的逆向关系

## 知识点

- 小数加减混合运算
- 解决两步以上的小数运算实际问题
- 简单的小数运算估算

## 教学情境（成都化改写）

学校运动会 50 米接力赛，四（2）班四个同学的成绩：

| 同学 | 第 1 棒 | 第 2 棒 | 第 3 棒 | 第 4 棒 |
|---|---|---|---|---|
| 小明 | 8.6 秒 | - | - | - |
| 小红 | - | 9.2 秒 | - | - |
| 小刚 | - | - | 8.8 秒 | - |
| 小芳 | - | - | - | 9.4 秒 |

四（3）班四个同学的总成绩为 36.5 秒。

问题串：
- 问题 1：四（2）班总用时是多少？
- 问题 2：哪个班赢了？快多少秒？
- 问题 3：如果小明再快 0.3 秒，四（2）班能领先多少？

## 课堂讨论点

- 为什么用时少的反而是赢家？（逆向关系）
- 估算："8.6 + 9.2 + 8.8 + 9.4" 大约是多少？怎么快速估？

## 学生易错点

- 比较结果时方向搞反（以为大数赢）
- 多步运算中漏算或重复算
- 加减混合时运算顺序错

## 巩固练习

1. 6.8 + 7.2 - 3.5 = ?
2. 10 - (4.6 + 2.7) = ?
3. 一根绳子长 12.5 米，剪去两段，分别长 3.8 米和 4.6 米，还剩多少米？
