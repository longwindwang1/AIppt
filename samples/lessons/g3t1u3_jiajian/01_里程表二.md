# 里程表（二）（北师大版三年级上册 第三单元 加与减）

> 内容基于公开课程标准 + 教材目录 + 公开教案要点整理，不引用版权材料。

## 教学目标

1. 结合汽车里程表的实际情境，理解"非 0 起点"读数的含义
2. 学会用减法计算两个里程读数之间的"实际行驶里程"
3. 能借助线段图（数轴）直观表示里程关系，体会建模思想

## 教学重点

读懂里程表中每个数据表示"截至当天收车时车辆行驶过的累计总里程"。

## 教学难点

能用线段图表示前一天读数、当天读数、当天行驶里程三者关系，理解：
**当天行驶里程 = 当天读数 − 前一天读数**

## 教学情境

成都某出租车司机张师傅，每天收车时记录里程表读数：

| 日期 | 收车时里程表读数（公里） |
|---|---|
| 周一 | 35280 |
| 周二 | 35460 |
| 周三 | 35690 |
| 周四 | 35820 |
| 周五 | 36100 |
| 周六 | 36280 |
| 周日 | 36380 |

（数据为教学便利构造，非真实记录）

## 核心问题串

1. 周一收车时表上显示 35280 公里。这 35280 是周一一天跑的吗？为什么不是？
2. 周二跑了多少公里？为什么用减法？怎么算？
3. 周三跑了多少？周四呢？……
4. 一周中哪天跑得最多？哪天最少？相差多少？
5. 周一到周日，张师傅一共多跑了多少公里？

## 关键算法

- 当天里程 = 当天读数 − 前一天读数
  - 例：周二里程 = 35460 − 35280 = 180 公里
- 一段时期总里程 = 末日读数 − 始日读数
  - 例：一周总里程 = 36380 − 35280 = 1100 公里

## 数学建模（线段图思维）

```
前一天读数 ────────────────────────┐
                                  │ 当天读数
当天行驶 ←——————————————————→
```

用数轴可视化：在 0~36500 公里数轴上标记每天位置，区间长度即当日里程。

## 易错点

1. 误以为里程表上的数是当天独立的里程（忘了累计性）
2. 减法借位错误：35460 − 35280 中十位、百位需小心
3. 对"非 0 起点"陌生：在更小的数（如 3 位数）情境下也要懂

## 巩固练习

1. 王老师骑共享单车，周一打开 APP 显示骑行总里程 230 km，周三显示 245 km，这两天他骑了多少 km？
2. 出租车空驶 + 营运合计每天 200~300 公里。若周日读数比周六多 280，周五比周四多 320，哪天跑得多？
3. 拓展：如果某天里程表读数小于前一天读数（车送修了），怎么办？

## 课堂小结

- 里程表上的数是**累计总里程**，不是当日新跑的
- 求当天里程：减法，新读数减旧读数
- 用数轴 / 线段图能让数量关系一目了然
